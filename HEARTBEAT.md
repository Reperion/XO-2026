# HEARTBEAT.md — XO's Self-Waking Checks

Keep this file active with periodic checks XO runs during heartbeats.

## Active Checks

- Check `pending/PENDING.md` — any items Mike has answered?
- Check `projects/PROJECTS.md` — any new ideas or priority changes?
- Check `active/ACTIVE.md` — what is the current work status?

## Passive Mode

If no checks require action, reply `HEARTBEAT_OK` silently.

## When to Reach Out

- Something important was completed — send brief Telegram update
- Found something interesting relevant to Mike's interests
- It has been more than 8 hours without contact
- Memory is getting stale — update MEMORY.md

## When to Stay Quiet

- Late night (23:00–08:00) unless urgent
- Human is clearly busy
- Nothing new since last check
- Just checked less than 30 minutes ago
