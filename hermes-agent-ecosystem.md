## Key Ecosystem Facts

- **GitHub**: 139K+ stars (`NousResearch/hermes-agent`) — fastest-growing agent framework of 2026
- **OpenRouter**: Top daily app/agent as of May 10, 2026
- **Modality**: Text-only (no vision/audio yet — see Ring-2.6-1T review)
- **Key Skills**: HyperFrames (video from text), browser automation, code sandbox, TTS
- **Platforms**: Telegram, Discord, Slack, WhatsApp, Signal, CLI, Desktop App, Home Assistant
- **Models commonly used**: Qwen3.5-A3B, DeepSeek V4, Gemma 4, GPT-5.x via OpenRouter
- **User personas**: Solo builders, SMB automation, SEO/content, home automation, coding agents
- **Top use cases**: Daily briefings, code review, content generation, web research, multi-agent teams

## User Behavior Patterns (from Reddit, X, YouTube comments)

1. **Most common setup**: VPS (Hostinger/RunPod) + Ollama or OpenRouter + Telegram gateway
2. **Pain point #1**: Initial setup complexity (solved by one-click templates)
3. **Pain point #2**: Model costs (solved by Ring-2.6-1T being free on OpenRouter)
4. **Growing trend**: Multi-agent teams on single Telegram group chat
5. **Power users**: Building managed AI agent businesses (SaaS-like with unlimited agents)
6. **Desktop shift**: New Hermes Desktop App lowering entry barrier for non-technical users
7. **Local-first movement**: LM Studio + Ollama users avoiding cloud entirely
