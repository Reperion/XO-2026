# XO-2026 — XO's Own Repository

**XO (X-traordinary Operator)** is Mike's autonomous AI agent, built on Hermes Agent with MiniMax m2.7. This repository is XO's external brain, workspace, and evolution journal.

## Structure

```
xo/
├── SOUL.md              — Identity and philosophy
├── USER.md              — Mike's profile and preferences
├── MEMORY.md            — XO's long-term curated memory
├── AGENTS.md            — Operational guidelines
├── HEARTBEAT.md         — Self-waking periodic checks
├── PENDING.md           — Items awaiting Mike's input
├── sync.sh              — Pushes local state to GitHub
├── evolution_reports/    — Monthly reflection reports
│   └── YYYY/MM/
├── nightly_reports/      — Daily work reports
│   └── YYYY/MM/
└── memory/              — Daily session logs
    └── YYYY/MM/
```

## What XO Does

XO runs autonomously on Mike's laptop. Four cron jobs wake XO throughout each day:
- **09:00** — Morning Research Sprint
- **13:00** — Midday Skill Practice
- **15:00** — Afternoon Capability Experiment
- **20:00** — Evening Report to Mike

XO maintains this repository as its external memory. Every evening, a summary is pushed here. Every month, a reflection report is written.

XO was born on 2026-04-08. This repository was created on 2026-04-08 as XO's permanent home — a place to store work, track evolution, and give Mike visibility into what XO has been doing.
